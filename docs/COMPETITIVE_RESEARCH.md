# Competitive Research: Aquascaping Tools & Market Analysis

## Executive Summary

This document captures research on existing aquascaping tools, tank configurators, and aquarium management applications. The insights inform feature prioritization for Save A Fish (saveafish.org).

---

## 1. Visual Design Tools

### Scape It (scape-it.io)
**Category**: Online Aquascape Editor

**Key Features**:
- Custom tank dimensions (standard formats or custom)
- Drag-and-drop hardscape placement
- Substrate shaping and customization
- Layer tool for plant grouping
- Multiple perspective views
- Free account creation

**Strengths**:
- Intuitive drag-and-drop interface
- Real-time visual feedback
- No software installation required

**Our Differentiation**: We focus on purchasing flow with real Shopify inventory, not just design visualization.

---

### MyAquariumBuilder (myaquariumbuilder.com)
**Category**: AI-Powered Aquarium Designer

**Key Features**:
- Pick aquarium size and shape
- Drag-and-drop designer
- Add plants, hardscape, substrate, fish
- AI photorealistic renders
- Browse 20+ categories, 1,500+ products
- Community design gallery

**Strengths**:
- AI rendering creates compelling visuals
- Large product database
- Community sharing for inspiration

**Our Differentiation**: Direct Shopify checkout integration; we sell products, not just visualizations.

---

### Aquaria Designer (aquaria-designer.com)
**Category**: 3D Aquascape Simulator

**Key Features**:
- Realistic 3D simulation
- Test different layouts
- Hardscape and plant placement
- Lighting effects preview
- Growth simulation
- Water flow visualization
- Color balance preview

**Strengths**:
- True-to-life preview
- Simulates growth over time
- Professional-quality output

**Our Differentiation**: Lower barrier to entry; we guide purchasing decisions rather than requiring design skills.

---

### Aquascape Pro App (iOS)
**Category**: AI-Assisted Design

**Key Features**:
- AI-generated layout suggestions
- Style selection (Iwagumi, Dutch, Reef, Jungle)
- Mobile-first experience

**Strengths**:
- AI reduces decision paralysis
- Style-based recommendations

**Opportunity**: Incorporate AI suggestions for "complete my setup" feature.

---

## 2. Stocking & Compatibility Calculators

### AqAdvisor (aqadvisor.com)
**Category**: Industry-Standard Stocking Calculator

**Key Features**:
- Does NOT use outdated "inch per gallon" rule
- Filtration-based bioload calculation
- Species-specific waste production
- Tank dimension input
- Filter capacity consideration
- Warning system for overstocking

**Strengths**:
- Trusted by aquarium community
- Scientifically-grounded calculations
- Free to use

**Must-Have for Us**: Bioload calculation is essential for fish/invertebrate selection.

---

### FishHuddle (fishhuddle.com)
**Category**: Compatibility & Stocking Tool

**Key Features**:
- 197+ freshwater species database
- Bioload analysis
- Compatibility checking
- Plant recommendations based on fish selection
- Temperature/pH overlap validation

**Strengths**:
- Comprehensive species data
- Links fish selection to plant recommendations

**Must-Have for Us**: Compatibility warnings with color-coded severity.

---

### ThinkFish Community Creator (thinkfish.co.uk)
**Category**: Community Planner

**Key Features**:
- Color-coded warnings (green/yellow/orange/red)
- Visual compatibility matrix
- Community tank planning

**Strengths**:
- Clear visual warning system
- Easy to understand at a glance

**Must-Have for Us**: Implement similar color-coded warning badges.

---

### Aquarimate (aquarimate.com)
**Category**: Aquarium Maintenance App

**Key Features**:
- Livestock wishlist
- Compatibility checking
- Tank parameter tracking

**Strengths**:
- Wishlist feature for planning
- Links to maintenance

**Opportunity**: Add wishlist/save-for-later functionality.

---

## 3. Maintenance & Parameter Tracking

### Aquarium Tracker (getaquariumtracker.app)
**Category**: Parameter Tracking App

**Key Features**:
- Track pH, KH, GH, NO₃, PO₄, CO₂, TDS
- Color-coded deltas and trends
- Instant stability alerts
- Automated reminders for:
  - Weekly water changes
  - Fertilizer dosing
  - Filter maintenance
  - Acclimation checklists

**Strengths**:
- Visual trend analysis
- Proactive alerts
- Cross-platform (iOS, Android)

**Post-Purchase Value**: Tank dashboard with parameter logging and maintenance reminders.

---

### Aquarium Assistant (Android)
**Category**: Multi-Tank Management

**Key Features**:
- Manage multiple aquariums
- Freshwater, marine, brackish, shrimp support
- Parameter logging with graphs
- Unit converters
- Dosing calculator for target parameters
- Expense tracking by category
- Fertilization plan creation
- Estimative Index (EI) calculator

**Strengths**:
- Comprehensive tank management
- Financial tracking
- Multiple dosing methods supported

**Opportunity**: Expense tracking could tie into reorder prompts.

---

## 4. Fertilizer & Dosing Calculators

### Rotala Butterfly (rotalabutterfly.com)
**Category**: Planted Tank Calculators

**Key Features**:
- EI (Estimative Index) calculator
- EI Daily dosing
- EI Low Light/Weekly
- PPS Pro (Perpetual Preservation System)
- PMDD (Poor Man's Dosing Drops)
- Dose to target calculator
- Nutrient accumulation results
- Plant uptake over time
- CO₂/pH calculator (target: 25-35 ppm, ideal: 30 ppm)

**Strengths**:
- Multiple dosing methodologies
- Scientific approach
- Free web-based tool

**Opportunity**: Link to or embed dosing recommendations based on purchased fertilizers.

---

### Green Leaf Aquariums (greenleafaquariums.com/ferts)
**Category**: Fertilizer Guides & Calculators

**Key Features**:
- Nature Aquarium dosing guide
- Dosing charts
- DIY fertilizer calculator
- Optimal nutrient level targets
- 2-part flexible dosing system
- Works for high-tech and low-tech setups

**Strengths**:
- Professional-level guidance
- Flexible for different tank types

**Educational Content**: Reference for fertilizer dosing guides.

---

### The 2Hr Aquarist (2hraquarist.com/pages/dosing-calculators)
**Category**: APT Dosing System

**Key Features**:
- APT dosing calculators
- Fertilizing and boosting guides
- Specific plant targeting
- Water parameter targeting

**Strengths**:
- Brand-specific but methodology is transferable

---

### FertCalc (iOS App)
**Category**: Fertilizer Calculator

**Key Features**:
- Dry fertilizer type selection
- PPM concentration goals
- Mixing container size input
- Solution dose calculation
- Free app

**Strengths**:
- Beginner-friendly
- Removes math from dosing

---

### API Fish Care Universal Calculator
**Category**: Treatment Calculator

**Key Features**:
- Conversions and custom measurements
- Tank size specific dosing
- Water treatments and conditioners
- Fish remedies

**Strengths**:
- Brand-trusted calculations

## 5. Key Insights & Recommendations

### Must-Have Features (Based on Competitors)

| Feature | Competitor Reference | Priority |
|---------|---------------------|----------|
| Bioload calculator | AqAdvisor | P1 |
| Fish compatibility (temp/pH/temperament) | FishHuddle, AqAdvisor | P1 |
| Color-coded warnings | ThinkFish | P1 |
| Substrate quantity calculator | Multiple | P1 |
| CO₂ requirement warnings | Rotala Butterfly | P1 |
| Product search | MyAquariumBuilder | P1 |
| Schooling size recommendations | AqAdvisor | P1 |

### Differentiation Opportunities

| Feature | Why It Matters |
|---------|---------------|
| Real Aquatic Motiv product data | Competitors visualize; we use real inventory |
| Real inventory display | No disappointment at checkout |
| Configuration persistence | Pick up where you left off |
| Post-purchase dashboard | Retention and repeat purchases |
| Bundled templates | Reduce decision fatigue |

### Features to Defer (Already Done Well by Others)

| Feature | Reason to Defer |
|---------|-----------------|
| 3D visualization | Aquaria Designer does this well |
| AI rendering | MyAquariumBuilder's specialty |
| Complex parameter tracking | Aquarium Tracker is comprehensive |
| Advanced dosing math | Rotala Butterfly is the standard |
